DISHcovery

"DISHcovery" - is a web application, which takes list of available ingredients in either text or image format and generates recipes that can be prepared using those ingredients. This website can be very useful for beginners and those who are bored of cooking same dishes everyday. Frontend: HTML, CSS, Javascript Backend: Python, Flask Database: MongoDB AI Models used: Convolusional Neural Networks(CNN) and Cosine Similarity
